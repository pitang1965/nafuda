# Milestone v1.0: MVP

**Status:** ✅ SHIPPED 2026-04-28

## Overview

なふだ v1.0 MVP。QRコードでその場でSNSリンクを共有・コネクションを記録できる推し活向けデジタル名刺Webアプリ。Phase 0でフロントエンドUX検証、Phase 1で認証・プロフィール基盤、Phase 2でイベントチェックイン、Phase 3でQR・コネクション・PWA対応を完成させた。

## Milestone Summary

**Key Decisions:**
- TanStack Start を採用（Vite ベース・型安全ルーティング・Cloudflare Workers 公式サポート）
- Neon Postgres（neon-http）+ Drizzle ORM（型安全なクエリ）
- Better Auth（Google / Facebook OAuth、cookieCache 無効化で Cloudflare Workers 対応）
- QR URL スキーム: `/u/{urlId}/p/{shareToken}`（urlId が公開識別子、shareToken は参照検証用）
- コネクションは one-way 設計（双方向は 2 行）・イベントコンテキストを非正規化保存
- PWA: バニラ JS Service Worker（Workbox バンドル問題を回避）

**Issues Resolved:**
- TanStack Router のキャッチオール問題（/api/auth/* バイパス）
- Better Auth の TanStack Start API 差異（createServerFn / createAPIFileRoute）
- SW が npm インポートをバンドルせず動作しない問題 → バニラ JS に書き換え
- SW が HTML をキャッシュして SSR をバイパスしタイトル・viewport が壊れる問題

**Technical Debt:**
- Hyperdrive（Cloudflare Workers Paid プラン）未導入（wrangler.toml にコメントアウト）
- `<link rel="manifest">` / viewport 等の HTML meta は直接 RootDocument に記述（HeadContent SSR 未動作）
