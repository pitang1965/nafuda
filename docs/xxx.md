## 3. プロフィール作成ウィザード

**URL:** `/profile/wizard`  **認証:** 必要

初回ログイン後にペルソナが未作成の場合に表示。5ステップ構成。

| ステップ | 内容 |
| --- | --- |
| 1. URL-ID | 公開識別子の入力・空き確認（英数字・ハイフン・アンダースコア、4〜20文字） |
| 2. 表示名 | ハンドルネーム入力（必須、50文字以下） |
| 3. 推しタグ | 推しタグ入力（1個以上必須） |
| 4. アバター | アバター画像URL入力 or 自動生成（イニシャルアバター） |
| 5. 完了 | 作成完了 → `/me` へ遷移 |

## 5. プロフィール編集

**URL:** `/profile/edit`  **認証:** 必要

| 項目名 | UIパーツ | 仕様 |
| --- | --- | --- |
| アバター | 画像 / InitialsAvatar | アバターURLに連動してリアルタイム更新 |
| アバターURL | テキストボックス | 任意。未入力時はイニシャルアバター |
| 表示名 | テキストボックス | 必須・50文字以下 |
| bio | テキストエリア | 任意・200文字以下 |
| 推しタグ | OshiTagInput | emblor によるタグ入力。追加・削除可 |
| 同担拒否 | チェックボックス | ON = 参加者一覧から非表示 |
| SNSリンク一覧 | カードリスト | プラットフォーム選択・URL入力・並び順変更・削除 |
| SNSプラットフォーム | セレクト | x / instagram / tiktok / youtube / discord / line_openchat / github / spotify / other |
| リンクを追加ボタン | ボタン | 末尾に空のリンクカードを追加 |
| 保存ボタン | ボタン | 各フィールドを個別に保存（autosave 形式） |
