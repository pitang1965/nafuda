## 1. ホーム画面（Home）

| 項目名 | UIパーツ | 仕様 | 必須 |
| --- | --- | --- | --- |

### 1-2. ランディング（未ログイン時）

| 項目名 | UIパーツ | 仕様 | 必須 |
| --- | --- | --- | --- |

## 2. イベントルーム画面（EventRoom）

**URL:** `/event/:id`

| 項目名 | UIパーツ | 仕様 | 必須 |
| --- | --- | --- | --- |
| 戻るボタン | ボタン | `/` へ遷移 | ○ |
| イベント名 | テキスト（h1） | モックデータのイベント名を表示 | ○ |
| 日時・場所 | テキスト | 📍アイコン＋日付＋会場名 | ○ |
| QR共有ボタン | ボタン | 押下でQRモーダルを表示 | |
| 参加者カウント | テキスト | "参加中 N / 全体M 名" を表示。カード表示に合わせてカウントアップ | ○ |
| 参加者カード一覧 | カードリスト | 300msごとに1枚ずつアニメーション表示。タップで `/p/:id` へ遷移 | ○ |
| 　└ ハンドルネーム | テキスト | 参加者のハンドルネームを表示 | ○ |
| 　└ 推しタグ | バッジ一覧 | 推し名をバッジとして表示（最大3件） | ○ |
| 　└ ジャンルタグ | バッジ一覧 | ジャンルをバッジとして表示（最大2件） | ○ |
| 　└ 同担ポリシーアイコン | アイコン | 同担歓迎 🙆 / 同担拒否 🙅 / 未設定は非表示。カード右隅に小さく表示 | |
| ローディングテキスト | テキスト | 全カード表示完了前に "つながっています…" を表示 | |
| 完了テキスト | テキスト | 全カード表示後に "✓ 全員とつながりました" を表示 | |
| QRモーダル | モーダル | イベントURLのQRコードを表示。閉じるで非表示 | |

> **備考:** idが `demo` 以外の場合は `/` にリダイレクト

## 4. なふだ編集画面（ProfileEdit）

**URL:** `/me/edit`

| 項目名 | UIパーツ | 仕様 | 必須 |
| --- | --- | --- | --- |
| 戻るボタン | ボタン | `/me` へ遷移 | ○ |
| 保存ボタン | ボタン（ghost） | 押下で "保存済み ✓" に2秒間変化 | ○ |
| アバター | 絵文字 / イニシャル | アバター絵文字入力に連動してリアルタイム更新 | ○ |
| アバター絵文字 | テキストボックス | 任意の絵文字1文字。未入力時はハンドルネームのイニシャルで代替 | |
| ハンドルネーム | テキストボックス | なふだのハンドルネーム編集（必須） | ○ |
| 推しタグ | タグ入力 | 推し名を追加・削除。サジェスト付き自由記述 | |
| ジャンルタグ | タグ入力 | ジャンルを追加・削除。サジェスト付き自由記述 | |
| 同担ポリシー | ラジオ / セレクト | 同担歓迎 / 同担拒否 / 未設定（デフォルト） | |
| 自己紹介 | テキストエリア | bio の編集（4行） | |
| リンク一覧 | カードリスト | 各リンクのラベル・URL・表示切替・削除を管理 | |
| 　└ ラベル | テキストボックス | リンク名（例: X, Instagram） | |
| 　└ URL | テキストボックス | リンクURL | |
| 　└ 表示切替 | トグルボタン | ONでなふだに表示、OFFで非表示 | |
| 　└ 削除ボタン | アイコンボタン（🗑） | リンクを一覧から削除 | |
| リンクを追加ボタン | ボタン | 末尾に空のリンクカードを追加 | |

> **備考:** 本名・会社名・役職はプロト UI に露出させない。ハンドルネーム前提の設計。

## 5. なふだ詳細画面（ProfileDetail）

**URL:** `/p/:id`

| 項目名 | UIパーツ | 仕様 | 必須 |
| --- | --- | --- | --- |
| 戻るボタン | ボタン | 前の画面へ戻る | ○ |
| ログアウトボタン | ボタン（ghost） | ログイン時のみ表示 | |
| アバター | 絵文字 / イニシャル | 設定した絵文字アバター、未設定時はイニシャル | ○ |
| ハンドルネーム | テキスト（h1） | なふだのハンドルネーム | ○ |
| 同担ポリシー | バッジ | 同担歓迎 🙆 / 同担拒否 🙅 。設定がある場合のみ表示 | |
| 推しタグ一覧 | バッジ一覧 | 登録した推し名。設定がある場合のみ表示 | |
| ジャンルタグ一覧 | バッジ一覧 | 登録したジャンル。設定がある場合のみ表示 | |
| つながり情報 | テキスト | ログイン済み＋つながり済みの場合のみ表示。「○○で出会った」形式のイベント名または日時 | |
| 自己紹介 | テキスト | ログイン時のみ表示。設定がある場合のみ表示 | |
| 参加した現場 | カードリスト | ログイン時のみ表示。チェックイン済みの現場履歴 | |
| リンク一覧 | リンクカード | ログイン時のみ表示。設定がある場合のみ表示。外部リンクとして開く | |
| 自分のメモ | テキストエリア / テキスト | ログイン済み＋つながり済みの場合のみ表示。タップで編集モードへ | |
| メモ保存ボタン | ボタン（primary） | 編集モード時に表示 | |
| メモキャンセルボタン | ボタン（ghost） | 編集モード時に表示 | |
| つながりに追加ボタン | ボタン（primary） | ログイン済み＋未つながりの場合に表示 | |
| つながりを削除ボタン | ボタン（ghost・赤） | ログイン済み＋つながり済みの場合に表示 | |
| ログインして見るボタン | ボタン（primary） | 未ログイン時に表示 | |
| ログインゲートモーダル | モーダル | 未ログイン状態でアクセスした際に自動表示 | |

> **備考:** 未ログイン時は自己紹介・参加した現場・リンクは 🔒 ロックセクションとして表示され非表示

## 6. つながり一覧画面（Connections）

**URL:** `/connections`

| 項目名 | UIパーツ | 仕様 | 必須 |
| --- | --- | --- | --- |
| 戻るボタン | ボタン | `/` へ遷移 | ○ |
| 画面タイトル | テキスト | "つながり一覧" | ○ |
| 総人数バッジ | バッジ | 全つながり件数を表示（例: 5人） | ○ |
| 検索フィールド | テキストボックス（search） | ハンドルネーム・推し・ジャンルで絞り込み（大文字小文字無視） | |
| フィルターチップ | ボタン（複数選択不可） | すべて / 🎤 現場経由 / 👤 直接つながり で絞り込み | |
| 表示件数 | テキスト | 絞り込み後の件数を "N人表示中" で表示 | ○ |
| つながりカード | カードリスト | つながり日時の降順で表示。タップで `/p/:id` へ遷移 | ○ |
| 　└ アバター | 絵文字 / イニシャル | 設定した絵文字アバター、未設定時はイニシャル | ○ |
| 　└ ハンドルネーム | テキスト | なふだのハンドルネーム | ○ |
| 　└ 推しタグ | バッジ一覧 | 登録した推し名（最大2件） | |
| 　└ つながり種別タグ | タグ | 現場経由の場合は 🎤 現場名、直接の場合は "直接" | ○ |
| 　└ つながり日時 | テキスト | 年月日形式（例: 2025年1月1日） | ○ |
| 　└ メモプレビュー | テキスト | メモが存在する場合のみ表示 | |
| 該当なしテキスト | テキスト | 絞り込み結果が0件の場合に "該当する人が見つかりません" を表示 | |
